# Student-Grade-Management-System
A visual student grade management tool developed with Python + Tkinter + Matplotlib. It supports the entry, search, deletion, modification, sorting, statistics, saving, and loading of student information (student ID, name, and scores in multiple subjects). The tool provides a GUI interface and a scrollable view to adapt to massive data, intuitively displays the score distribution of each subject through boxplots, and realizes efficient management and visual analysis of score data.
